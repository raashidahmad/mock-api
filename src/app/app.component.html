Help on Queries

Search an entity through Id
http://localhost:3000/entityName/id
Search an entity through other attributes
http://localhost:3000/entityName?attr=value
Sort result against a column in a specific order
http://localhost:3000/siteCharts?_sort=column&_order=asc/desc
Pagination in a query
http://localhost:3000/siteCharts?_start=number&_end=number
